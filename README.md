# donjons-et-delices
Projet de session d'été 2024 à l'Université de Sherbrooke, fait avec Samuel Boisvert et Simon Major.
Fait avec Unreal Engine 5, Blender pour le modèle et animations du joueur, et assest gratuits sur le Unreal Engine Marketplace et en ligne.
